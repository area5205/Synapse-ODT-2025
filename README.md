# Synapse-ODT-2025
From Idea to Impact- Open Design &amp; Technology Final Showcase by Arya and Anoushka

This Installation is an attempt to explore how technology often tries to replicate human connection by responding to their emotions via algorithms.
 Itfollows the principles of English sentence formation to give the user an advisory or observational response to their emotion along with the visual element of a pop-out card which can then be interpreted by the user in various ways.
